# Mandoo Shop

## 프로젝트 소개
Mandoo Shop은 JS 강의를 마무리하면서 간단한 협업을 체험해보기 위해 제작된 프로젝트 입니다.

## 사용한 기술 스택
- HTML
- CSS
- JavaScript

## 페이지 소개
### 메인 페이지: 쇼핑몰의 홈 페이지 입니다.
### 로그인/회원가입 페이지: 사용자의 회원 가입 및 로그인 기능을 구현했습니다.
### 상품 목록 페이지: 상품 목록을 무한 스크롤을 이용해 구현했습니다.
### 장바구니 페이지: 상품 목록 페이지에서 담은 상품과 가격을 확인할 수 있습니다.
### 주문 페이지: 상품 목록 페이지에서 구매하기 버튼을 클릭하면 이동되며, 상품을 주문할 수 있습니다.

## 프로젝트 구조
```
/
├── cart/              # 장바구니 페이지
├── imgs/              # 상품 이미지 폴더
├── login/             # 로그인 페이지
├── order/             # 주문 페이지
├── products/          # 상품 목록 페이지
├── signup/            # 회원가입 페이지
├── .gitignore
├── db.json            # 상품 데이터
├── index.html         # 메인 페이지
├── README.md
├── reset.css
├── script.js
└── style.css
```
